FOUNT FOR LINUX (PORTABLE)

Thank you for downloading the portable version of Fount, the minimal Fountain screenplay editor.

QUICK START:
1. Extract the tarball.
2. Open a terminal in the folder.
3. Grant execution permission: chmod +x fount
4. Run: ./fount

Optional: To open a specific script:
./fount my_screenplay.fountain

CONFIGURATION:
Fount stores settings in: ~/.config/fount/fount.conf

SHORTCUTS:
- Ctrl+S: Save
- Ctrl+H: Scene Navigator (Story Map)
- Ctrl+L: Character Sidebar
- / (Slash): Open command bar (type /help)

For more information, visit: https://github.com/BeetleBot/Fount
