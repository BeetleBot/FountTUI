# Fount for Windows

This is the portable binary of Fount, a terminal-based screenplay editor.

## Installation
To run Fount:
1. Extract the `fount.zip` file.
2. Inside, you will find `fount.exe`.
3. Double-click it to launch!

## Features
- Full terminal integration.
- Custom Zen Studio theme support.
- Windows-native clipboard support.

## Troubleshooting
If Windows blocks the file during the first launch:
1. Right-click `fount.exe`.
2. Select "Properties".
3. Check "Unblock" at the bottom-right corner.
4. Click OK and run it again.
